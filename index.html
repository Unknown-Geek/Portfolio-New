Software Projects

Software Projects

01 / 05

Code Recet 2.0 Website

CODERECET 2.0 is a 36-hour hackathon organized by the College of Engineering Trivandrum that brings students together to and build solutions to real-world problems. I contributed to the project’s frontend, implementing an intuitive, responsive interface that enhanced usability and collaboration during the event.

02 / 05

PgBee Website

Contributed to the frontend of PgBee, a student accommodation platform that connects students with verified PGs and hostels. Built a clean, user-friendly interface enabling effortless search, booking, and transparent rent & bill management. Enhanced the overall user experience by implementing responsive design and smooth navigation.

03 / 05

Tech Sprouts Website

I contributed to the frontend of Tech Sprouts, a student-led platform that makes technology learning fun for school students through coding, design, and creative projects. The initiative empowers young minds from grades 5–12 to explore, create, and innovate with technology. It has also been recognized as a Top 20 startup at the Indian Startup League 2025.

04 / 05

Story Generator

Developed Story Generator, an AI-powered web app that creates unique stories from uploaded images. Users can upload multiple images, select genre and narration preferences, and experience real-time audio storytelling with visual highlighting. Built with React, Tailwind, Flask, and Gemini API to deliver a seamless, responsive experience.

05 / 05

Space Invaders - OpenCV

Built Space Invaders Remastered, a modern take on the classic arcade game with AI-powered hand tracking for intuitive controls. Integrated a high score system using SQLite, along with sound effects, scrolling backgrounds, and a leaderboard for an engaging experience. Designed to run seamlessly with webcam input, delivering a nostalgic yet innovative gameplay experience.
